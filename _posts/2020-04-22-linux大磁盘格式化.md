```
1，超过2T的大磁盘没法直接fdisk进行格式化分区，需要用parted进行格式化分区：
    查看磁盘（fdisk -l）---分区---格式化（创建文件系统）----挂载（自动挂载）
	

2，安装parted	
    yum install parted
	
	
3. 查看还未挂载的磁盘
[root@localhost ~]# lsblk
NAME            MAJ:MIN RM  SIZE RO TYPE MOUNTPOINT
sda               8:0    0   10G  0 disk 
├─sda1            8:1    0    1G  0 part /boot
└─sda2            8:2    0    9G  0 part 
  ├─centos-root 253:0    0    8G  0 lvm  /
  └─centos-swap 253:1    0    1G  0 lvm  [SWAP]
sdb               8:16   0    2T  0 disk 
sr0              11:0    1  4.2G  0 rom  
[root@localhost ~]# parted /dev/sdb

unit UNIT  set the default unit to UNIT
version  displays the current version of GNU Parted and copyright information

(parted)              输入mklabel            //改变分区表的格式
新的磁盘标签类型？     输入gpt           //设置为gpt的分区格式
(parted)              输入mkpart
分区名称？ []?             输入/dev/sdb    //填写分区名称
文件系统类型？ [ext2]?        输入ext3  //填写格式化类型
起始点？            输入0%   //代表是从第一个磁道开始的
结束点？            输入100%    //是代表到最后一个磁道
(parted)           输入print free
Model: VMware, VMware Virtual S (scsi)
Disk /dev/sdb: 2190GB
Sector size (logical/physical): 512B/512B
Partition Table:    输入gpt  //修改为gpt格式
Number  Start  End  Size  File system  Name 标志
1  17.4kB  2190GB  2190GB  /dev/sdb1
(parted)           输入quit  //退出


5,格式化系统
[root@localhost ]# mkfs.ext3  /dev/sdb   ##将/dev/sdb磁盘分区格式化为exts  centos7默认是xfs

6,创建挂载点
[root@localhost ]# mkdir /data

7，挂载
[root@localhost ]# mount /dev/sdb /data

```
