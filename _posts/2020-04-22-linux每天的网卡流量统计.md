```
因为很多vps或者服务器都是限流量的，但是又很多服务商并没有提供详细的流量表，比如每天的流量表，所以肯定有人很想知道自己服务器到底跑了多少流量。
	

1， 安装
	yum install epel-release -y && yum install -y vnstat

2. 查看网卡
	ifconfig

3. 创建监控数据库
	vnstat -u -i eth0
 
4. 启动服务并设置开机启动
	service vnstat start
	chkconfig vnstat on
	
5. 流量查看命令
	1. 看每天的流量统计命令
		vnstat -d
	2. 看每月的流量统计命令
		vnstat -m
	
```
