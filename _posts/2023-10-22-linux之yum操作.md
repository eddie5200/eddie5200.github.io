```
1. 安装软件：
	yum install packageName；

2. 升级全部软件：
	yum update；

3. 升级某个软件：
	yum update packageName；

4. 检查哪些软件可以升级：
	yum check-update；

5. 卸载某个软件包：
	yum remove packageName 或 yum erase packageName；

6. 列出所有已安装的软件：
	yum list installed；

7. 列出所有可以更新升级的软件
	yum list updates；

8. 列出某个包的详细信息：
	yum info packageName；

9. 查找某个软件包：
	yum search packageName；


10. 清除所有 YUM 缓存：
	yum clean all；

11. 显示已启用的软件源：
	yum repolist； （repo id 前如果有叹号，表示该仓库的元数据过期了）

12. 显示所有软件源：
	yum repolist all；

13.	查看某个软件包的依赖：
	yum deplist packageName；

14.	查找某个命令来自哪个软件包：
	yum provides commandName 或 yum whatprovides commandName
	
```
